# Security Policy

At ToggleMesh, we take security very seriously. We appreciate your efforts to responsibly disclose your findings, and we will make every effort to acknowledge your contributions.

## Supported Versions

Currently, the following versions of ToggleMesh are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| >= v1.0.0-rc | ✅                  |
| < v1.0.0     | ❌                  |

## Reporting a Vulnerability

If you discover a security vulnerability within ToggleMesh, please **DO NOT** open a public issue. 

Instead, please send an email directly to the maintainer at **sdwcktarakanov@gmail.com**.

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- The version of ToggleMesh you are evaluating.

We will review your report within 48 hours and provide a timeline for a patch. Once the vulnerability is resolved, we will release a security advisory. Thank you for helping keep ToggleMesh safe for everyone!
